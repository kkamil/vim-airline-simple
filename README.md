## vim-airline-simpler

Theme for vim-airline: https://github.com/bling/vim-airline based on simple theme.

### How to install

Clone repository

```
cd ~/.vim/bundle
git clone https://github.com/kkamil/vim-airline-simpler.git
```

Add `let g:airline_theme = 'simpler'` to your vimrc.
